## BUAA CO 课程用 CPU 评测机

### 说明

这些评测机能够生成测试数据以评测自行搭建的 CPU。不过我们不能保证数据具有足够的强度，所以也请自行构造测试数据来检验。

P3 的评测机仅支持对拍，P4 及之后的评测机可实现“自测”。

每一个 P 的评测机的构造大体相同，但细节上又有所不同。使用前请仔细阅读对应目录下的 `readme.md` 文件。如果在使用中出现了任何难以解决的问题，可以通过邮件 `24371491@buaa.edu.cn` 联系。

### 鸣谢

感谢 Tony 学长等人开发的魔改版 Mars！它大大减少了我们实现评测的工作量。

### 局限性

P3 的评测机数据较弱，且并不总是合法，容易出现死循环的状况。建议用 P4 及以后评测机目录下的 `generator.py` 文件来替换 P3 目录下的同名文件。
